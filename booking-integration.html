Instrucciones de Integración

🏆 Recomendación Gold Standard

Para Centro Médico Universal recomendamos Zoho Booking porque:

  • ✅ Maneja múltiples doctores y especialidades
  • ✅ Integración con seguros médicos
  • ✅ Historial de pacientes
  • ✅ HIPAA compliant (privacidad médica)
  • ✅ Reportes y análisis

Pasos de Implementación:

  1. Crear Cuenta en Zoho Booking
    Visita zoho.com/bookings y crea una cuenta con el email del centro médico
  2. Configurar Servicios
    Añade cada especialidad como un "servicio" con duración y precio
  3. Agregar Personal
    Crea perfiles para cada doctor con sus horarios disponibles
  4. Personalizar Formulario
    Añade campos para seguro médico, síntomas, etc.
  5. Obtener Código de Integración
    Ve a Settings → Embed → Copy code
  6. Integrar en tu Sitio
    Pega el código en tu index.html donde quieras el calendario

💡 Tip Pro

Configura webhooks para enviar datos de citas a tu sistema de gestión hospitalaria (HIS) automáticamente.
